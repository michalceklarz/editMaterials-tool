# editMaterialsREADME # 

This program has been written for Windows 10 and must be run using a Windows command as well as by clicking on the icon of the file editMaterials.exe.

This program adds and edits material information of IfcWalls and IfcSlabs contained in the IFC model (schema Ifc4X1).  The input argument is a path to the IFC file that is to be edited. The output argument is the edited IFC model with updated changes. 

## Example command ##

>> cd "C:/Path/to/the/folder/with/the/executable"
>> editMaterials.exe
